# What this app is

This is an **AI QA / Evaluation Lab**. It is meant to:

* store and version prompts
* create datasets and test cases
* run prompts across one or more AI providers
* compare outputs across models
* grade outputs automatically and manually
* inspect execution traces
* detect regressions
* review and approve results
* manage workspaces and permissions securely

This is not a student demo. It should behave like a serious engineering product.
